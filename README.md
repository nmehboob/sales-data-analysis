# Sales Data Analysis Tool

A simple Python program to analyze sales data and generate reports.

## Features
- Calculate total and average sales
- Find highest and lowest sales days
- Calculate median sales
- Visual sales trend chart
- Performance status indicator

## How to Use

### Option 1: Manual Entry
```bash
python sales_analysis.py
```
Then enter your daily sales data

### Option 2: Use Example Data
```bash
python example_data.py
```
